Montichello Coffee

Responsive website built as an exam project. The goal was to create a fully adaptive multi-section website with interactive UI elements, optimized performance, and clean structure.

Live Demo

https://natalinamatioshko.github.io/exam-2-montichello/

Overview

Montichello Coffee is a modern landing page that highlights coffee culture through visual storytelling, galleries, and interactive components.
The project includes a hero slider, content sections, a dynamic gallery, smooth animations, and a custom Google Map.

Features

Responsive layout for desktop, tablet and mobile

Hero image slider (Slick Slider)

Coffee Journal carousel with adaptive breakpoints

Interactive 3D effect cards using Atropos.js

Lightbox gallery with captions

Custom Google Map marker

Burger menu for mobile navigation

Preloader animation

LocalStorage form submission

Smooth scrolling and clean transitions

Tech Stack

HTML5

SCSS (variables, mixins, separate partials)

JavaScript (modular scripts for slider, gallery, map, UI interactions)

jQuery (for sliders and utilities)

Slick Slider

Atropos.js

Lightbox2

Google Maps API
